# Memorama-Saludable
🍎 Memorama de Alimentos Saludables para Preescolar 🥦
¡Bienvenido al Memorama de Alimentos Saludables! Este es un recurso educativo digital e interactivo diseñado especialmente para niños en etapa preescolar (3 a 6 años). A través del clásico juego de memoria, los niños desarrollan habilidades cognitivas como la concentración y la retención visual, mientras aprenden de forma divertida la importancia de comer comida saludable.

✨ Características Principales
Diseño Kawaii e Infantil: Ilustraciones amigables dibujadas directamente con vectores SVG (Manzana, Plátano, Zanahoria, Brócoli, Fresa, Pescado, Leche y Huevo) con colores pastel vibrantes y tipografías redondeadas y legibles.
Voz Educativa (Text-to-Speech): El juego lee en voz alta un consejo divertido sobre el beneficio del alimento cada vez que el niño encuentra una pareja (ej. "¡Zanahoria! Es buena para que tus ojos vean mejor").
Efectos de Sonido Sintetizados: Utiliza la Web Audio API del navegador para generar tonos musicales en tiempo real al voltear, acertar, fallar y ganar. ¡No requiere descargar archivos de audio externos!
Dificultad Progresiva:
Fácil: 4 parejas (8 cartas). Ideal para los más pequeños (3 años).
Medio: 6 parejas (12 cartas). Para niños de 4-5 años.
Difícil: 8 parejas (16 cartas). Para niños de 5-6 años.
Celebración Bouncy: Animación de confeti de colores y modal de premiación interactivo que otorga estrellas (1 a 3 ⭐) según los intentos realizados.
Guía para Padres y Maestros: Panel expandible en el pie de página con información sobre los beneficios de cada alimento para apoyar la labor docente y familiar.
Accesibilidad y Compatibilidad: Diseño responsivo para pantallas móviles o tabletas, y compatibilidad con teclado (Enter y Espacio) y lectores de pantalla.
📁 Estructura del Proyecto
index.html: Estructura del tablero, marcadores de estrellas, modal de victoria y guía informativa.
styles.css: Diseño responsivo, rejillas dinámicas (Grid), animaciones de acierto/error y giros 3D optimizados para móviles.
app.js: Lógica del juego (mezclado de cartas, control de estados, audio sintético, motor de voz en español y confeti).
instrucciones_de_uso.md: Manual detallado para jugar y usar las opciones de sonido y voz.
planeacion_didactica.md: Planeación de clase pedagógica estructurada para maestros de preescolar.
todo.md: Hoja de ruta y estado de las tareas del proyecto.
🚀 Cómo Jugar
No necesitas instalar nada en tu computadora ni contar con conexión a internet.

Abre la carpeta del proyecto.
Haz doble clic sobre el archivo index.html para abrirlo en tu navegador de internet favorito (Google Chrome, Microsoft Edge, Safari o Firefox).
Selecciona el nivel de dificultad, haz clic sobre una estrella/carta ¡y que empiece la diversión saludable!
Nota: La primera vez que entres, haz clic en cualquier parte de la pantalla para permitir que el navegador reproduzca los sonidos y la voz de los alimentos.
